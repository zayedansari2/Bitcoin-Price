# Bitcoin-Price
Real time Bitcoin price USD


<img src="https://github.com/user-attachments/assets/9fab3cc1-4804-4c8c-bc21-51bf58234eac" alt="image" style="max-width:100%; height:auto;" />


This program fetches the real-time Bitcoin price using the CoinDesk API. By parsing JSON data from CoinDesk's API, it displays the current Bitcoin price in USD, GBP, and EUR. It's a simple and effective way to track Bitcoin prices using Java.
